Scripts for professional workflows, business analysis, and project management in the COSMOS framework.

## Available Scripts

### Report Generation
- `report-generator.py` - Generate professional reports
- `presentation-builder.sh` - Create presentations from data

### Market Analysis
- `market-analyzer.py` - Market opportunity analysis
- `competitive-analysis.py` - Competitive landscape analysis

### IP Management
- `patent-tracker.py` - Patent portfolio tracking
- `ip-analyzer.sh` - Intellectual property analysis

### Project Management
- `project-dashboard.py` - Project status dashboard
- `milestone-tracker.sh` - Milestone and deliverable tracking

### Business Intelligence
- `metrics-calculator.py` - Calculate business metrics (TRL, patent potential, etc.)
- `roi-estimator.py` - Return on investment estimation

## Usage Examples

```bash
# Generate a professional report
python report-generator.py --type quarterly --output report.pdf

# Analyze market opportunity
python market-analyzer.py --sector energy --technology atomic-ex-light

# Track patents
python patent-tracker.py --portfolio cosmos --status active

# Calculate project metrics
python metrics-calculator.py --project rafaelia --metric TRL
```

## Requirements

- Python 3.8+
- Pandas, NumPy
- ReportLab (for PDF generation)
- Matplotlib (for charts)

Install:
```bash
pip install pandas numpy reportlab matplotlib
```
