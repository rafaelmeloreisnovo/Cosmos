# Frontend Scripts

Web UI and visualization scripts for COSMOS framework.

## Available Scripts

### Setup & Development
- `setup.sh` - Setup frontend development environment
- `dev-server.sh` - Start development server
- `build.sh` - Build production bundle

### Visualization Components
- `fibonacci-visualizer.js` - Fibonacci sequence visualization
- `galaxy-viewer.js` - Galaxy pattern viewer
- `fractal-renderer.js` - Fractal geometry renderer
- `energy-dashboard.js` - Energy conversion dashboard

### UI Components
- `cosmos-ui-kit.js` - Reusable UI component library
- `data-grid.js` - Data table component
- `chart-components.js` - Chart and graph components

## Technology

- React 18+
- D3.js for data visualization
- Three.js for 3D graphics
- Material-UI for components

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Components

All components support the COSMOS design system with cosmic themes and scientific visualization standards.
