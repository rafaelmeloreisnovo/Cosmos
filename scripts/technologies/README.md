## Usage

Each script includes detailed usage instructions. Example:


## Dependencies

- Python 3.8+
- NumPy, SciPy, Matplotlib (for Python scripts)
- Bash 4.0+
- ImageMagick (for visualization scripts)

Install Python dependencies:
```bash
pip install -r requirements.txt
```
