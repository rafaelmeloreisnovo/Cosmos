### 3.2 Compression Theory

#### Information-Theoretic Limits

**Shannon's Source Coding Theorem:**
```
L ≥ H(X)
```
Average code length cannot be less than entropy.

**Rate-Distortion Theory:**
For lossy compression, tradeoff between rate R and distortion D.

**Bibliographic References:**
- Cover, T. M., & Thomas, J. A. (2006). *Elements of Information Theory* (2nd ed.). Wiley.
- Sayood, K. (2017). *Introduction to Data Compression* (5th ed.). Morgan Kaufmann.

#### Existing Compression Algorithms

**Lossless:**
1. Huffman coding: Optimal prefix codes
2. LZ77/LZ78: Dictionary-based
3. Arithmetic coding: Near-optimal
4. Bzip2: Block-sorting
5. LZMA: High compression ratio

**Lossy:**
1. JPEG: DCT-based image compression
2. MP3: Psychoacoustic model
3. H.264/H.265: Video compression

**Bibliographic References:**
