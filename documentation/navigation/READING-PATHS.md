## Navigation Tips

### Efficient Reading Strategies:

1. **Progressive Disclosure:** Start with summaries, dive deeper as interest warrants
2. **Cross-References:** Follow links for related concepts
3. **Skim First:** Get overview before detailed reading
4. **Take Notes:** Document questions and insights
5. **Return Later:** Not all content needs immediate deep reading

### Document Structure:

All area documents follow similar structure:
1. Overview & Concepts
2. Academic Context
3. Analysis & Assessment
4. Validation Requirements
5. Recommendations
6. References

### Finding Specific Information:

- **Definitions:** Check GLOSSARY (if available) or relevant area document Section 1
- **Statistics:** Check INDEX Metrics section or FACTS document
- **Academic Sources:** EXTERNAL Bibliography
- **Repository Claims:** INTERNAL References or source files
- **Innovation Details:** PATENT-POTENTIAL or DISCOVERIES
- **Validation Status:** DISCOVERIES or specific area documents
